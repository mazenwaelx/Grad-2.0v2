# Logo Image Setup

## Instructions

Please place your Estasheer logo image in this directory with the following specifications:

### File Details
- **File name:** `estasheer-logo.png`
- **Recommended format:** PNG (for transparency support)
- **Recommended dimensions:** 400-600px width
- **Aspect ratio:** Maintain the original aspect ratio of your logo

### Your Logo
The logo features:
- Golden scales of justice integrated with the letter 'E'
- "Estasheer" text in navy blue
- Clean, professional legal branding

### Where It Will Appear
1. **Chat Empty State** - When no messages exist (280px max width)
2. **Login Page** - Above the login form (200px max width)
3. **Signup Page** - Above the signup form (200px max width)

### Responsive Sizing
The logo will automatically scale for different screen sizes:
- **Desktop:** Full size (200-280px)
- **Tablet:** Medium size (220px)
- **Mobile:** Small size (160-180px)

### Image Optimization Tips
- Use PNG format for transparent background
- Optimize file size (ideally under 200KB)
- Use high DPI/retina resolution for crisp display (2x or 3x)
- Ensure the logo has proper padding/whitespace around it

## After Adding the Logo

Once you've placed `estasheer-logo.png` in this directory, the application will automatically use it throughout the interface with proper animations and styling.

### Animations Applied
- **Float effect:** Gentle up-and-down motion (3s loop)
- **Scale-in:** Smooth entrance animation on page load
- **Drop shadow:** Golden glow effect matching the theme

### Fallback
If the image file is missing, you'll see a broken image icon. Make sure to place the file with the exact name: `estasheer-logo.png`
