# =====================================================
# Installation Notes
# =====================================================
# Basic installation:
#   pip install -r requirements-clean.txt
#
# For GPU support with CUDA 12.6:
#   pip install torch==2.11.0+cu126 --index-url https://download.pytorch.org/whl/cu126
#
# For production deployment:
#   pip install gunicorn
#   gunicorn -w 4 -k uvicorn.workers.UvicornWorker api_server:app
#
# Troubleshooting:
#   - If Tesseract fails, install from: https://github.com/UB-Mannheim/tesseract/wiki
#   - If pyodbc fails, install ODBC Driver 17 for SQL Server
#   - For pdf2image, install Poppler: https://github.com/oschwartz10612/poppler-windows/releases/
